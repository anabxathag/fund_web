Web developer

A web developer is a programmer who specializes in, or is specifically engaged in, the development of World Wide Web applications using a client-sever model. The applications typically use HTML, CSS and JavaScript in the client, PHP, ASP.NET (C#), Python or Java in the client-sever, and http for communications between client and sever. A web content management system is often used to develop and maintain web applications.
